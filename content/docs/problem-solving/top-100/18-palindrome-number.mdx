### <TranslatedText en="Approach 1: Reverse the Number" bn="পদ্ধতি ১: সংখ্যা উল্টান" />

<TranslatedText
  en={<p>Reverse the entire number and compare it with the original.</p>}
  bn={<p>সম্পূর্ণ সংখ্যা উল্টান এবং মূলের সাথে তুলনা করুন।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

bool isPalindrome(int n) {
    // Negative numbers are not palindromes
    if (n < 0) {
        return false;
    }

    int original = n;
    long long reversed = 0;

    while (n > 0) {
        int digit = n % 10;
        reversed = reversed * 10 + digit;
        n /= 10;
    }

    return original == reversed;
}

int main() {
    int num1 = 12321;
    int num2 = 12345;

    cout << num1 << " is " << (isPalindrome(num1) ? "a palindrome" : "not a palindrome") << endl;
    cout << num2 << " is " << (isPalindrome(num2) ? "a palindrome" : "not a palindrome") << endl;

    return 0;
}
```

**Output:**
```
12321 is a palindrome
12345 is not a palindrome
```

```javascript
function isPalindrome(n) {
    // Negative numbers are not palindromes
    if (n < 0) {
        return false;
    }

    let original = n;
    let reversed = 0;
```

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;

bool isPalindromeString(int n) {
    // Negative numbers are not palindromes
    if (n < 0) {
        return false;
    }

    string str = to_string(n);
    string reversed = str;
    reverse(reversed.begin(), reversed.end());

    return str == reversed;
}

int main() {
    int num1 = 12321;
    int num2 = 12345;

    cout << num1 << " is " << (isPalindromeString(num1) ? "a palindrome" : "not a palindrome") << endl;
    cout << num2 << " is " << (isPalindromeString(num2) ? "a palindrome" : "not a palindrome") << endl;

    return 0;
}
```

### <TranslatedText en="Approach 3: Half Reverse (Optimized)" bn="পদ্ধতি ৩: হাফ রিভার্স (অপ্টিমাইজড)" />

<TranslatedText
  en={<p>Only reverse half the number to save operations. Compare first half with reversed second half.</p>}
  bn={<p>অপারেশন সংরক্ষণ করতে শুধুমাত্র অর্ধেক সংখ্যা উল্টান। প্রথম অর্ধেককে উল্টানো দ্বিতীয় অর্ধেকের সাথে তুলনা করুন।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

bool isPalindromeOptimized(int n) {
    // Negative numbers and numbers ending in 0 (except 0 itself)
    if (n < 0 || (n % 10 == 0 && n != 0)) {
        return false;
    }

    int reversedHalf = 0;

    // Reverse only half the number
    while (n > reversedHalf) {
        reversedHalf = reversedHalf * 10 + n % 10;
        n /= 10;
    }

    // For even length: n == reversedHalf
    // For odd length: n == reversedHalf / 10 (middle digit doesn't matter)
    return n == reversedHalf || n == reversedHalf / 10;
}

int main() {
    int num1 = 12321;
    int num2 = 12345;
    int num3 = 1221;

    cout << num1 << " is " << (isPalindromeOptimized(num1) ? "a palindrome" : "not a palindrome") << endl;
    cout << num2 << " is " << (isPalindromeOptimized(num2) ? "a palindrome" : "not a palindrome") << endl;
    cout << num3 << " is " << (isPalindromeOptimized(num3) ? "a palindrome" : "not a palindrome") << endl;

    return 0;
}
```

**Output:**
```
12321 is a palindrome
12345 is not a palindrome
1221 is a palindrome
```

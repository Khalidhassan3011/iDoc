## <TranslatedText en="Problem Statement" bn="সমস্যা বিবৃতি" />

<TranslatedText
  en={
    <>
      <p><strong>Definition:</strong> For an n-digit number, if the sum of each digit raised to power n equals the number itself, it's a Narcissistic number.</p>
      <p><strong>Examples:</strong></p>
      <ul>
        <li>153 = 1³ + 5³ + 3³ = 1 + 125 + 27 = 153 ✓ (Narcissistic)</li>
        <li>9474 = 9⁴ + 4⁴ + 7⁴ + 4⁴ = 6561 + 256 + 2401 + 256 = 9474 ✓</li>
        <li>123 = 1³ + 2³ + 3³ = 1 + 8 + 27 = 36 ✗ (Not Narcissistic)</li>
      </ul>
    </>
  }

### <TranslatedText en="Approach 1: Count Digits First" bn="পদ্ধতি ১: প্রথমে ডিজিট গণনা করুন" />

<TranslatedText
  en={<p>First count the number of digits, then calculate the sum of each digit raised to that power.</p>}
  bn={<p>প্রথমে ডিজিটের সংখ্যা গণনা করুন, তারপর প্রতিটি ডিজিট সেই শক্তিতে উত্থাপিত যোগফল গণনা করুন।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <cmath>
using namespace std;

bool isNarcissistic(int num) {
    if (num < 0) return false;

    int original = num;
    int digitCount = 0;

    // Count digits
    int temp = num;
    while (temp > 0) {
        digitCount++;
        temp /= 10;
    }

    // Calculate sum of digits raised to power of digit count
    int sum = 0;
    temp = num;
    while (temp > 0) {
        int digit = temp % 10;
        sum += pow(digit, digitCount);
        temp /= 10;
    }

    return sum == original;
}

int main() {
    int numbers[] = {0, 1, 9, 153, 370, 371, 407, 1634, 9474, 123};

    for (int num : numbers) {
        cout << num << " is "
             << (isNarcissistic(num) ? "a Narcissistic number" : "not a Narcissistic number")
             << endl;
    }

    return 0;
}
```

```javascript
function isNarcissistic(num) {
    if (num < 0) return false;

    const original = num;
    const digitCount = num.toString().length;

    // Calculate sum of digits raised to power of digit count
    let sum = 0;
    let temp = num;
    while (temp > 0) {
        const digit = temp % 10;
        sum += Math.pow(digit, digitCount);
        temp = Math.floor(temp / 10);
    }

    return sum === original;
}

let numbers = [0, 1, 9, 153, 370, 371, 407, 1634, 9474, 123];

for (let num of numbers) {
    console.log(`${num} is ${isNarcissistic(num) ? 'a Narcissistic number' : 'not a Narcissistic number'}`);
}
```

```cpp
#include <iostream>
#include <string>
#include <cmath>
using namespace std;

bool isNarcissisticString(int num) {
    if (num < 0) return false;

    string numStr = to_string(num);
    int digitCount = numStr.length();
    int sum = 0;

    for (char ch : numStr) {
        int digit = ch - '0';
        sum += pow(digit, digitCount);
    }

    return sum == num;
}

int main() {
    int numbers[] = {0, 1, 9, 153, 370, 371, 407, 1634, 9474, 123};

    for (int num : numbers) {
        cout << num << " is "
             << (isNarcissisticString(num) ? "a Narcissistic number" : "not a Narcissistic number")
             << endl;
    }

    return 0;
}
```

```python
def is_narcissistic_string(num):
    if num < 0:
        return False

    num_str = str(num)
    digit_count = len(num_str)
    total = sum(int(digit) ** digit_count for digit in num_str)

    return total == num

numbers = [0, 1, 9, 153, 370, 371, 407, 1634, 9474, 123]
```

```javascript
function isNarcissisticString(num) {
    if (num < 0) return false;

    const numStr = num.toString();
    const digitCount = numStr.length;

    const sum = numStr
        .split('')
        .reduce((acc, digit) => acc + Math.pow(parseInt(digit), digitCount), 0);

    return sum === num;
}

let numbers = [0, 1, 9, 153, 370, 371, 407, 1634, 9474, 123];

for (let num of numbers) {
    console.log(`${num} is ${isNarcissisticString(num) ? 'a Narcissistic number' : 'not a Narcissistic number'}`);
}
```

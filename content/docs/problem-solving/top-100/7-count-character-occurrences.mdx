## <TranslatedText en="Approach 1: Using Hash Map / Dictionary" bn="পদ্ধতি ১: হ্যাশ ম্যাপ / ডিকশনারি ব্যবহার করে" />

<TranslatedText
  en="Use a hash map (object/dictionary) to store character counts. Most efficient approach."
  bn="অক্ষর গণনা সংরক্ষণের জন্য একটি হ্যাশ ম্যাপ (অবজেক্ট/ডিকশনারি) ব্যবহার করুন। সবচেয়ে দক্ষ পদ্ধতি।"
/>

<Tabs items={['C++', 'Python', 'JavaScript']}>
<Tab value="C++">
```cpp
#include <iostream>
#include <unordered_map>
#include <string>
using namespace std;

unordered_map<char, int> countCharacters(const string& str) {
    unordered_map<char, int> charCount;

    for (char ch : str) {
        charCount[ch]++;
    }

    return charCount;
}

void printCharacterCounts(const string& str) {
    unordered_map<char, int> counts = countCharacters(str);

    cout << "Character counts for \"" << str << "\":" << endl;

    for (const auto& pair : counts) {
        cout << pair.first << ": " << pair.second;
        if (pair.second == 1) {
            cout << " time" << endl;
        } else {
            cout << " times" << endl;
        }
    }
}

int main() {
    string name = "Shirsh";

    printCharacterCounts(name);

    return 0;
}
```

function countCharacters(str) {

function countCharactersMap(str) {
    const charCount = new Map();

## <TranslatedText en="Approach 3: Array-based Counting (ASCII Characters)" bn="পদ্ধতি ৩: অ্যারে-ভিত্তিক গণনা (ASCII অক্ষর)" />

<TranslatedText
  en="Use an array indexed by ASCII values for faster access (works only for ASCII characters)."
  bn="দ্রুত অ্যাক্সেসের জন্য ASCII মান দ্বারা সূচিত একটি অ্যারে ব্যবহার করুন (শুধুমাত্র ASCII অক্ষরের জন্য কাজ করে)।"
/>

<Tabs items={['C++', 'Python', 'JavaScript']}>
<Tab value="C++">
```cpp
#include <iostream>
#include <string>
using namespace std;

void countCharactersArray(const string& str) {
    // Array to store count for all ASCII characters (0-127)
    int charCount[128] = {0};

    // Count each character
    for (char ch : str) {
        charCount[(int)ch]++;
    }

    // Print only characters that appeared
    cout << "Character counts for \"" << str << "\":" << endl;

    for (int i = 0; i < 128; i++) {
        if (charCount[i] > 0) {
            cout << (char)i << ": " << charCount[i];
            cout << (charCount[i] == 1 ? " time" : " times") << endl;
        }
    }
}

int main() {
    string name = "Shirsh";

    countCharactersArray(name);

    return 0;
}
```
</Tab>

function countCharactersArray(str) {
    // Array for all ASCII characters (0-127)
    const charCount = new Array(128).fill(0);

    for (let ch of str) {

        if (charCount[i] > 0) {
            const char = String.fromCharCode(i);

                console.log(`${char}: ${charCount[i]} time`);
            } else {
                console.log(`${char}: ${charCount[i]} times`);

    }
}

## <TranslatedText en="Approach 4: Case-Insensitive Counting" bn="পদ্ধতি ৪: কেস-ইনসেনসিটিভ গণনা" />

<TranslatedText
  en="Count characters ignoring case (treat 'S' and 's' as the same character)."
  bn="কেস উপেক্ষা করে অক্ষর গণনা করুন ('S' এবং 's' কে একই অক্ষর হিসাবে বিবেচনা করুন)।"
/>

<Tabs items={['C++', 'Python', 'JavaScript']}>
<Tab value="C++">
```cpp
#include <iostream>
#include <unordered_map>
#include <string>
#include <cctype>
using namespace std;

unordered_map<char, int> countCharactersCaseInsensitive(const string& str) {
    unordered_map<char, int> charCount;

    for (char ch : str) {
        // Convert to lowercase
        char lower = tolower(ch);
        charCount[lower]++;
    }

    return charCount;
}

void printCharacterCountsCaseInsensitive(const string& str) {
    unordered_map<char, int> counts = countCharactersCaseInsensitive(str);

    cout << "Character counts for \"" << str << "\" (case-insensitive):" << endl;

    for (const auto& pair : counts) {
        cout << pair.first << ": " << pair.second;
        cout << (pair.second == 1 ? " time" : " times") << endl;
    }
}

int main() {
    string name = "Shirsh";

    // Case-sensitive
    cout << "Case-sensitive count:" << endl;
    unordered_map<char, int> counts1 = countCharacters(name);
    for (const auto& p : counts1) {
        cout << p.first << ": " << p.second << endl;
    }

    cout << "\nCase-insensitive count:" << endl;
    printCharacterCountsCaseInsensitive(name);

    return 0;
}
```

function countCharactersCaseInsensitive(str) {
    const charCount = {};

    for (let ch of str) {
        // Convert to lowercase
        const lower = ch.toLowerCase();
        if (charCount[lower]) {
            charCount[lower]++;
        } else {
            charCount[lower] = 1;
        }
    }

    return charCount;
}

function printCharacterCountsCaseInsensitive(str) {
    const counts = countCharactersCaseInsensitive(str);

    console.log(`Character counts for "${str}" (case-insensitive):`);

    for (let char in counts) {
        const count = counts[char];
        if (count === 1) {
            console.log(`${char}: ${count} time`);
        } else {
            console.log(`${char}: ${count} times`);
        }
    }
}

// Test
const name = "Shirsh";

console.log("Case-sensitive count:");
printCharacterCounts(name);

console.log("\nCase-insensitive count:");
printCharacterCountsCaseInsensitive(name);

## <TranslatedText en="Comparison of Approaches" bn="পদ্ধতির তুলনা" />

| Approach | Time | Space | Best For |
|----------|------|-------|----------|
| Hash Map | O(n) | O(k) | **Best for interviews - flexible and efficient** |
| Built-in (Counter/Map) | O(n) | O(k) | Production code, cleaner syntax |
| Array (ASCII) | O(n) | O(1) | ASCII-only, performance-critical |
| Case-insensitive | O(n) | O(k) | When case doesn't matter |

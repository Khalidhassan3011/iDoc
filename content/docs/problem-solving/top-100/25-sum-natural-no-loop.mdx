### <TranslatedText en="Approach 1: Mathematical Formula (Gauss's Formula)" bn="পদ্ধতি ১: গাণিতিক সূত্র (গাউসের সূত্র)" />

<TranslatedText
  en={<p>Use the famous formula discovered by Carl Friedrich Gauss: Sum = n(n+1)/2</p>}
  bn={<p>কার্ল ফ্রিডরিখ গাউস আবিষ্কৃত বিখ্যাত সূত্র ব্যবহার করুন: যোগফল = n(n+1)/2</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

long long sumOfNaturalNumbers(int n) {
    // Using Gauss's formula: n * (n + 1) / 2
    return (long long)n * (n + 1) / 2;
}

int main() {
    int numbers[] = {10, 100, 1000, 5};

    for (int n : numbers) {
        long long sum = sumOfNaturalNumbers(n);
        cout << "Sum of first " << n << " natural numbers = " << sum << endl;
    }

    return 0;
}
```

**Output:**
```
Sum of first 10 natural numbers = 55
Sum of first 100 natural numbers = 5050
Sum of first 1000 natural numbers = 500500
Sum of first 5 natural numbers = 15
```

### <TranslatedText en="Approach 2: Using Arithmetic Progression Formula" bn="পদ্ধতি ২: গাণিতিক অগ্রগতি সূত্র ব্যবহার করে" />

<TranslatedText
  en={<p>Natural numbers form an arithmetic progression with first term a=1, common difference d=1, and n terms.</p>}
  bn={<p>প্রাকৃতিক সংখ্যা একটি গাণিতিক অগ্রগতি গঠন করে যার প্রথম পদ a=1, সাধারণ পার্থক্য d=1, এবং n পদ।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

long long sumArithmeticProgression(int n) {
    // AP formula: Sum = n/2 * (first term + last term)
    // For natural numbers: first = 1, last = n
    int first = 1;
    int last = n;

    return (long long)n * (first + last) / 2;
}

int main() {
    int n = 10;

    long long sum = sumArithmeticProgression(n);
    cout << "Sum of first " << n << " natural numbers = " << sum << endl;

    // Verify with direct formula
    long long directSum = (long long)n * (n + 1) / 2;
    cout << "Verification (direct formula) = " << directSum << endl;

    return 0;
}
```

**Output:**
```
Sum of first 10 natural numbers = 55
Verification (direct formula) = 55
```
  </Tab>

## <TranslatedText en="How Gauss Discovered the Formula" bn="গাউস কীভাবে সূত্র আবিষ্কার করেছিলেন" />

<TranslatedText
  en={
    <>
      <p><strong>The Story:</strong> Young Carl Friedrich Gauss amazed his teacher by instantly summing 1 to 100.</p>
      <p><strong>His Insight:</strong></p>
      <ul>
        <li>Write the sum forward: 1 + 2 + 3 + ... + 98 + 99 + 100</li>
        <li>Write the sum backward: 100 + 99 + 98 + ... + 3 + 2 + 1</li>
        <li>Add them pair-wise: (1+100) + (2+99) + (3+98) + ... = 101 each</li>
        <li>There are 100 pairs, each summing to 101</li>
        <li>Total = 100 × 101 = 10,100</li>
        <li>But we counted twice, so: 10,100 / 2 = 5,050</li>
      </ul>
      <p><strong>General Formula:</strong> n × (n+1) / 2</p>
    </>
  }
  bn={
    <>
      <p><strong>গল্প:</strong> তরুণ কার্ল ফ্রিডরিখ গাউস তার শিক্ষককে তাৎক্ষণিকভাবে 1 থেকে 100 পর্যন্ত যোগ করে অবাক করেছিলেন।</p>
      <p><strong>তার অন্তর্দৃষ্টি:</strong></p>
      <ul>
        <li>সামনের দিকে যোগফল লিখুন: 1 + 2 + 3 + ... + 98 + 99 + 100</li>
        <li>পিছনের দিকে যোগফল লিখুন: 100 + 99 + 98 + ... + 3 + 2 + 1</li>
        <li>জোড়া অনুযায়ী যোগ করুন: (1+100) + (2+99) + (3+98) + ... = প্রতিটি 101</li>
        <li>100টি জোড়া আছে, প্রতিটি 101 এ যোগ হয়</li>
        <li>মোট = 100 × 101 = 10,100</li>
        <li>কিন্তু আমরা দুইবার গণনা করেছি, তাই: 10,100 / 2 = 5,050</li>
      </ul>
      <p><strong>সাধারণ সূত্র:</strong> n × (n+1) / 2</p>
    </>
  }
/>

## <TranslatedText en="Mathematical Proof" bn="গাণিতিক প্রমাণ" />

<TranslatedText
  en={
    <>
      <p><strong>Method 1: Pairing</strong></p>
      <pre>
S = 1 + 2 + 3 + ... + (n-1) + n
S = n + (n-1) + (n-2) + ... + 2 + 1
-----------------------------------
2S = (n+1) + (n+1) + (n+1) + ... + (n+1) + (n+1)
2S = n × (n+1)
S = n × (n+1) / 2
      </pre>
      <p><strong>Method 2: Induction</strong></p>
      <ul>
        <li>Base case: n=1, Sum = 1 = 1(1+1)/2 ✓</li>
        <li>Assume true for n=k: Sum(k) = k(k+1)/2</li>
        <li>Prove for n=k+1: Sum(k+1) = Sum(k) + (k+1) = k(k+1)/2 + (k+1) = (k+1)(k+2)/2 ✓</li>
      </ul>
    </>
  }
  bn={
    <>

        <li>n=k এর জন্য সত্য ধরুন: Sum(k) = k(k+1)/2</li>
        <li>n=k+1 এর জন্য প্রমাণ করুন: Sum(k+1) = Sum(k) + (k+1) = k(k+1)/2 + (k+1) = (k+1)(k+2)/2 ✓</li>

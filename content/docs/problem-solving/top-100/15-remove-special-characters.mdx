### <TranslatedText en="Approach 1: Character-by-Character Check" bn="পদ্ধতি ১: অক্ষর-বাই-অক্ষর পরীক্ষা" />

  bn={<p>প্রতিটি অক্ষরের মধ্য দিয়ে পুনরাবৃত্তি করুন এবং শুধুমাত্র যেগুলি লোয়ারকেস অক্ষর (a-z) তা রাখুন।</p>}

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <string>
using namespace std;

string removeSpecialCharacters(string str) {
    string result = "";

    for (char ch : str) {
        // Check if character is lowercase letter (a-z)
        if (ch >= 'a' && ch <= 'z') {
            result += ch;
        }
    }

    return result;
}

int main() {
    string input = "hello@world!2024#code";

    cout << "Original: " << input << endl;
    cout << "Cleaned: " << removeSpecialCharacters(input) << endl;

    return 0;
}
```

**Output:**
```
Original: hello@world!2024#code
Cleaned: helloworldcode
```

```javascript
function removeSpecialCharactersBuiltin(str) {
    let result = "";

    for (let ch of str) {
        // Check if it's a lowercase letter using regex
        if (/[a-z]/.test(ch)) {
            result += ch;
        }
    }

    return result;
}

// Alternative: using filter and join
function removeSpecialCharactersModern(str) {
    return str.split('').filter(ch => /[a-z]/.test(ch)).join('');
}

let input = "hello@world!2024#code";

console.log(`Original: ${input}`);
console.log(`Cleaned (method 1): ${removeSpecialCharactersBuiltin(input)}`);
console.log(`Cleaned (modern): ${removeSpecialCharactersModern(input)}`);
```

### <TranslatedText en="Approach 3: Using Regular Expressions" bn="পদ্ধতি ৩: রেগুলার এক্সপ্রেশন ব্যবহার করে" />

<TranslatedText
  en={<p>Use regex to replace all non-lowercase characters in a single operation.</p>}
  bn={<p>একক অপারেশনে সব নন-লোয়ারকেস অক্ষর প্রতিস্থাপন করতে রেজেক্স ব্যবহার করুন।</p>}

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <string>
#include <regex>
using namespace std;

string removeSpecialCharactersRegex(string str) {
    // Remove all characters that are NOT lowercase letters
    regex pattern("[^a-z]");
    return regex_replace(str, pattern, "");
}

int main() {
    string input = "hello@world!2024#code";

    cout << "Original: " << input << endl;
    cout << "Cleaned: " << removeSpecialCharactersRegex(input) << endl;

    return 0;
}
```

**Output:**
```
Original: hello@world!2024#code
Cleaned: helloworldcode
```

```python
import re

def remove_special_characters_regex(s):

    return re.sub(r'[^a-z]', '', s)

input_str = "hello@world!2024#code"

print(f"Original: {input_str}")
print(f"Cleaned: {remove_special_characters_regex(input_str)}")
```

### <TranslatedText en="Approach 1: Using Temporary Variable" bn="পদ্ধতি ১: অস্থায়ী ভেরিয়েবল ব্যবহার করে" />

<TranslatedText
  en={<p>The most straightforward approach is to use a third temporary variable to hold one value during the swap.</p>}
  bn={<p>সবচেয়ে সহজ পদ্ধতি হল সোয়াপের সময় একটি মান ধরে রাখতে একটি তৃতীয় অস্থায়ী ভেরিয়েবল ব্যবহার করা।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 50;
    int b = 10;

    cout << "Before swap: a = " << a << ", b = " << b << endl;

    // Swap using temporary variable
    int temp = a;  // Store a in temp
    a = b;         // Assign b to a
    b = temp;      // Assign temp (original a) to b

    cout << "After swap: a = " << a << ", b = " << b << endl;

    return 0;
}
```

**Output:**
```
Before swap: a = 50, b = 10
After swap: a = 10, b = 50
```
  </Tab>


### <TranslatedText en="Approach 2: Using Arithmetic Operations (Without Temp)" bn="পদ্ধতি ২: গাণিতিক অপারেশন ব্যবহার করে (Temp ছাড়া)" />

<TranslatedText
  en={<p>We can swap without using a temporary variable by using addition and subtraction.</p>}
  bn={<p>যোগ এবং বিয়োগ ব্যবহার করে আমরা অস্থায়ী ভেরিয়েবল ছাড়াই সোয়াপ করতে পারি।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 50;
    int b = 10;

    cout << "Before swap: a = " << a << ", b = " << b << endl;

    // Swap without temporary variable
    a = a + b;  // a = 50 + 10 = 60
    b = a - b;  // b = 60 - 10 = 50
    a = a - b;  // a = 60 - 50 = 10

    cout << "After swap: a = " << a << ", b = " << b << endl;

    return 0;
}
```
  </Tab>
  <Tab value="Python">


### <TranslatedText en="Approach 3: Using XOR (Bitwise Operation)" bn="পদ্ধতি ৩: XOR ব্যবহার করে (বিটওয়াইজ অপারেশন)" />

<TranslatedText
  en={<p>XOR operation has a unique property: a ^ a = 0 and a ^ 0 = a. We can use this to swap without temporary variable or overflow issues.</p>}
  bn={<p>XOR অপারেশনের একটি অনন্য বৈশিষ্ট্য রয়েছে: a ^ a = 0 এবং a ^ 0 = a। আমরা এটি ব্যবহার করে অস্থায়ী ভেরিয়েবল বা ওভারফ্লো সমস্যা ছাড়াই সোয়াপ করতে পারি।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 50;
    int b = 10;

    cout << "Before swap: a = " << a << ", b = " << b << endl;

    // Swap using XOR
    a = a ^ b;  // a = 50 ^ 10
    b = a ^ b;  // b = (50 ^ 10) ^ 10 = 50
    a = a ^ b;  // a = (50 ^ 10) ^ 50 = 10

    cout << "After swap: a = " << a << ", b = " << b << endl;

    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python

```
  </Tab>
  <Tab value="JavaScript">

### <TranslatedText en="Approach 4: Using Language-Specific Features" bn="পদ্ধতি ৪: ভাষা-নির্দিষ্ট বৈশিষ্ট্য ব্যবহার করে" />

<TranslatedText
  en={<p>Modern programming languages provide elegant built-in ways to swap variables.</p>}
  bn={<p>আধুনিক প্রোগ্রামিং ভাষাগুলি ভেরিয়েবল সোয়াপ করার জন্য মার্জিত বিল্ট-ইন উপায় প্রদান করে।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <utility>  // for std::swap
using namespace std;

int main() {
    int a = 50;
    int b = 10;

    cout << "Before swap: a = " << a << ", b = " << b << endl;

    // Swap using std::swap
    swap(a, b);

    cout << "After swap: a = " << a << ", b = " << b << endl;

    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
a = 50
b = 10

print(f"Before swap: a = {a}, b = {b}")

# Pythonic way - tuple unpacking
a, b = b, a

print(f"After swap: a = {a}, b = {b}")
```
  </Tab>
  <Tab value="JavaScript">
```javascript
let a = 50;
let b = 10;

console.log(`Before swap: a = ${a}, b = ${b}`);

// ES6 destructuring
[a, b] = [b, a];

console.log(`After swap: a = ${a}, b = ${b}`);
```

</Tabs>

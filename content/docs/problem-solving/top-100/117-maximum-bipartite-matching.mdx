## <TranslatedText en="Solution" bn="সমাধান" />

<Tabs items={['Python']} groupId="language">

```python
def max_matching(graph, m, n):
    match = [-1] * n

    def dfs(u, seen):
        for v in range(n):
            if graph[u][v] and not seen[v]:
                seen[v] = True

                if match[v] == -1 or dfs(match[v], seen):
                    match[v] = u
                    return True
        return False

    result = 0
    for u in range(m):
        seen = [False] * n
        if dfs(u, seen):
            result += 1

    return result

# Example: m applicants, n jobs
graph = [[0, 1, 1, 0], [1, 0, 0, 1], [0, 0, 1, 0]]
print("Maximum matching:", max_matching(graph, 3, 4))
```

**Output:**
```
Maximum matching: 3
```

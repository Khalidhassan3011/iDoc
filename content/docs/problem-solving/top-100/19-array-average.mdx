### <TranslatedText en="Approach 1: Iterative Sum" bn="পদ্ধতি ১: পুনরাবৃত্তিমূলক যোগফল" />

<TranslatedText
  en={<p>Calculate the sum of all elements using a loop, then divide by the count.</p>}
  bn={<p>একটি লুপ ব্যবহার করে সমস্ত উপাদানের যোগফল গণনা করুন, তারপর গণনা দ্বারা ভাগ করুন।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <vector>
using namespace std;

double calculateAverage(vector<int>& arr) {
    if (arr.empty()) {
        return 0.0;
    }

    double sum = 0;
    for (int num : arr) {
        sum += num;
    }

    return sum / arr.size();
}

int main() {
    vector<int> arr = {10, 20, 30, 40, 50};

    double average = calculateAverage(arr);
    cout << "Average: " << average << endl;

    return 0;
}
```

**Output:**
```
Average: 30
```

```cpp
#include <iostream>
#include <vector>
#include <numeric>
using namespace std;

double calculateAverageBuiltin(vector<int>& arr) {
    if (arr.empty()) {
        return 0.0;
    }

    // Use accumulate to calculate sum
    double sum = accumulate(arr.begin(), arr.end(), 0.0);

    return sum / arr.size();
}

int main() {
    vector<int> arr = {10, 20, 30, 40, 50};

    double average = calculateAverageBuiltin(arr);
    cout << "Average: " << average << endl;

    return 0;
}
```

### <TranslatedText en="Approach 3: Recursive Solution" bn="পদ্ধতি ৩: রিকার্সিভ সমাধান" />

<TranslatedText

  bn={<p>শিক্ষাগত উদ্দেশ্যে রিকার্সিভভাবে যোগফল গণনা করুন (প্রোডাকশনের জন্য সুপারিশ করা হয় না)।</p>}

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <vector>
using namespace std;

double sumRecursive(vector<int>& arr, int index) {
    // Base case: reached end of array
    if (index >= arr.size()) {
        return 0;
    }

    // Recursive case: current element + sum of rest
    return arr[index] + sumRecursive(arr, index + 1);
}

double calculateAverageRecursive(vector<int>& arr) {
    if (arr.empty()) {
        return 0.0;
    }

    double sum = sumRecursive(arr, 0);
    return sum / arr.size();
}

int main() {
    vector<int> arr = {10, 20, 30, 40, 50};

    double average = calculateAverageRecursive(arr);
    cout << "Average (recursive): " << average << endl;

    return 0;
}
```

**Output:**
```
Average (recursive): 30
```

def sum_recursive(arr, index=0):
    # Base case: reached end of array

arr = [10, 20, 30, 40, 50]

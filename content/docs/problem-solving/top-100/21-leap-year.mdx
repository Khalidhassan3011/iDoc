### <TranslatedText en="Approach 1: Complete Logic with All Conditions" bn="পদ্ধতি ১: সমস্ত শর্ত সহ সম্পূর্ণ লজিক" />

<TranslatedText
  en={<p>Implement all three rules explicitly to determine if a year is a leap year.</p>}
  bn={<p>একটি বছর লিপ ইয়ার কিনা তা নির্ধারণ করতে সমস্ত তিনটি নিয়ম স্পষ্টভাবে বাস্তবায়ন করুন।</p>}

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

bool isLeapYear(int year) {
    // Divisible by 400 -> leap year
    if (year % 400 == 0) {
        return true;
    }

    // Divisible by 100 (but not 400) -> not leap year
    if (year % 100 == 0) {
        return false;
    }

    // Divisible by 4 (but not 100) -> leap year
    if (year % 4 == 0) {
        return true;
    }

    // Otherwise -> not leap year
    return false;
}

int main() {
    int years[] = {2024, 1900, 2000, 2023, 2100};

    for (int year : years) {
        cout << year << " is "
             << (isLeapYear(year) ? "a leap year" : "not a leap year")
             << endl;
    }

    return 0;
}
```

**Output:**
```
2024 is a leap year
1900 is not a leap year
2000 is a leap year
2023 is not a leap year
2100 is not a leap year
```

### <TranslatedText en="Approach 2: One-Line Condition" bn="পদ্ধতি ২: এক-লাইন শর্ত" />

<TranslatedText
  en={<p>Combine all conditions into a single boolean expression for conciseness.</p>}
  bn={<p>সংক্ষিপ্ততার জন্য সমস্ত শর্তকে একটি একক বুলিয়ান এক্সপ্রেশনে একত্রিত করুন।</p>}

```cpp
#include <iostream>
using namespace std;

bool isLeapYearOneLiner(int year) {
    return (year % 400 == 0) || (year % 4 == 0 && year % 100 != 0);
}

int main() {
    int years[] = {2024, 1900, 2000, 2023, 2100};

    for (int year : years) {
        cout << year << " is "
             << (isLeapYearOneLiner(year) ? "a leap year" : "not a leap year")
             << endl;
    }

    return 0;
}
```

```javascript
// JavaScript doesn't have built-in leap year check
// We use custom implementation
function isLeapYearCustom(year) {
    return (year % 400 === 0) || (year % 4 === 0 && year % 100 !== 0);
}

// Alternative: Check if Feb 29 exists
function isLeapYearDateCheck(year) {
    const feb29 = new Date(year, 1, 29);
    return feb29.getMonth() === 1; // Feb is month 1 (0-indexed)
}

let years = [2024, 1900, 2000, 2023, 2100];

console.log("Using custom logic:");
for (let year of years) {
    console.log(`${year} is ${isLeapYearCustom(year) ? 'a leap year' : 'not a leap year'}`);
}

console.log("\nUsing date check:");
for (let year of years) {
    console.log(`${year} is ${isLeapYearDateCheck(year) ? 'a leap year' : 'not a leap year'}`);
}
```

## <TranslatedText en="Understanding the Logic" bn="লজিক বোঝা" />

<TranslatedText
  en={
    <>
      <p><strong>Why these rules?</strong></p>
      <ul>
        <li><strong>Rule 1:</strong> Year divisible by 4 → Usually leap year (adds Feb 29)</li>
        <li><strong>Rule 2:</strong> Year divisible by 100 → NOT leap year (removes excess days over centuries)</li>
        <li><strong>Rule 3:</strong> Year divisible by 400 → IS leap year (fine-tunes the calendar)</li>
      </ul>
      <p>This system keeps our calendar aligned with Earth's orbit around the sun (365.2425 days).</p>
    </>
  }
  bn={
    <>
      <p><strong>এই নিয়মগুলি কেন?</strong></p>
      <ul>
        <li><strong>নিয়ম 1:</strong> 4 দ্বারা বিভাজ্য বছর → সাধারণত লিপ ইয়ার (ফেব 29 যোগ করে)</li>
        <li><strong>নিয়ম 2:</strong> 100 দ্বারা বিভাজ্য বছর → লিপ ইয়ার নয় (শতাব্দীর উপর অতিরিক্ত দিন সরিয়ে দেয়)</li>
        <li><strong>নিয়ম 3:</strong> 400 দ্বারা বিভাজ্য বছর → লিপ ইয়ার (ক্যালেন্ডার ফাইন-টিউন করে)</li>
      </ul>
      <p>এই সিস্টেম আমাদের ক্যালেন্ডারকে সূর্যের চারপাশে পৃথিবীর কক্ষপথের সাথে সারিবদ্ধ রাখে (365.2425 দিন)।</p>

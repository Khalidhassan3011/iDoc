## <TranslatedText en="Count Set Bits (1s)" bn="সেট বিট গণনা করা (১-এর সংখ্যা)" />

<TranslatedText
  en={<p>Count the number of 1s in binary representation efficiently using Brian Kernighan's algorithm.</p>}
  bn={<p>ব্রায়ান কার্নিগানের অ্যালগরিদম ব্যবহার করে বাইনারি প্রতিনিধিত্বে ১-এর সংখ্যা দক্ষতার সাথে গণনা করুন।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

// Iterative approach
int countSetBits(long n) {
    int count = 0;
    while (n) {
        count++;
        n &= (n - 1);  // Clear rightmost set bit
    }
    return count;
}

// Recursive approach
int countSetBitsRecursive(long n) {
    return (n ? 1 + countSetBitsRecursive(n & (n - 1)) : 0);
}

int main() {
    long n = 30;  // Binary: 11110
    cout << "Set bits in " << n << ": " << countSetBits(n) << endl;  // 4
    cout << "Set bits (recursive): " << countSetBitsRecursive(n) << endl;  // 4
    return 0;
}
```
  </Tab>
  <Tab value="Python">

/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int swapBytes(int x) {
    return (x << 8) | (x >> 8);
}

int main() {
    int x = 0x1234;
    int swapped = swapBytes(x);

    cout << "Original: 0x" << hex << x << endl;          // 0x1234
    cout << "Swapped:  0x" << hex << swapped << endl;    // 0x3412
    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
def swap_bytes(x):
    # Mask to keep only 16 bits
    return ((x << 8) | (x >> 8)) & 0xFFFF

x = 0x1234
swapped = swap_bytes(x)

print(f"Original: 0x{x:04x}")        # 0x1234
print(f"Swapped:  0x{swapped:04x}")  # 0x3412
```
  </Tab>
  <Tab value="JavaScript">

## <TranslatedText en="Using Arithmetic Operations" bn="গাণিতিক অপারেশন ব্যবহার করে" />

<TranslatedText
  en={<p>This method uses addition and subtraction to swap values. It works by storing the sum of both variables in one variable, then deriving the original values through subtraction.</p>}
  bn={<p>এই পদ্ধতি মান সোয়াপ করতে যোগ এবং বিয়োগ ব্যবহার করে। এটি একটি ভেরিয়েবলে উভয় ভেরিয়েবলের সমষ্টি সংরক্ষণ করে কাজ করে, তারপর বিয়োগের মাধ্যমে মূল মানগুলি প্রাপ্ত করে।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;
    int b = 10;

    a = a + b;  // a = 15
    b = a - b;  // b = 5
    a = a - b;  // a = 10

    cout << a << " " << b << endl; // 10 5
    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
a = 5

a = a + b  # a = 15
b = a - b  # b = 5
a = a - b  # a = 10

print(a, b)  # 10 5
```
  </Tab>
  <Tab value="JavaScript">
```javascript
let a = 5;

b = a - b;  // b = 5
a = a - b;  // a = 10

console.log(a, b); // 10, 5
```
  </Tab>
</Tabs>

## <TranslatedText en="Using XOR Operator" bn="XOR অপারেটর ব্যবহার করে" />

<TranslatedText
  en={<p>The XOR bitwise operator has a unique property: <code>a ^ a = 0</code> and <code>a ^ 0 = a</code>. This property allows us to swap values without any overflow issues that arithmetic operations might have.</p>}
  bn={<p>XOR বিটওয়াইজ অপারেটরের একটি অনন্য বৈশিষ্ট্য আছে: <code>a ^ a = 0</code> এবং <code>a ^ 0 = a</code>। এই বৈশিষ্ট্য আমাদের গাণিতিক অপারেশনের ওভারফ্লো সমস্যা ছাড়াই মান সোয়াপ করতে দেয়।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;
    int b = 10;

    a = a ^ b;
    b = a ^ b;
    a = a ^ b;

    cout << a << " " << b << endl; // 10 5
    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
a = 5
```

## <TranslatedText en="Using Language-Specific Features" bn="ভাষা-নির্দিষ্ট বৈশিষ্ট্য ব্যবহার করে" />

<TranslatedText
  en={<p>Modern programming languages offer built-in features for swapping values elegantly. While these technically may use temporary variables internally, they provide clean, readable syntax.</p>}
  bn={<p>আধুনিক প্রোগ্রামিং ভাষাগুলি মান সুন্দরভাবে সোয়াপ করার জন্য বিল্ট-ইন বৈশিষ্ট্য প্রদান করে। যদিও এগুলি প্রযুক্তিগতভাবে অভ্যন্তরীণভাবে অস্থায়ী ভেরিয়েবল ব্যবহার করতে পারে, তারা পরিষ্কার, পাঠযোগ্য সিনট্যাক্স প্রদান করে।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <utility>
using namespace std;

int main() {
    int a = 5;
    int b = 10;

    // Using std::swap
    swap(a, b);

    cout << a << " " << b << endl; // 10 5
    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
a = 5
b = 10

# Using tuple unpacking
a, b = b, a

print(a, b)  # 10 5
```
  </Tab>
  <Tab value="JavaScript">
```javascript
let a = 5;
let b = 10;

// Using destructuring
[a, b] = [b, a];

console.log(a, b); // 10, 5
```
  </Tab>
</Tabs>

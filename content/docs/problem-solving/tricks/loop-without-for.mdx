## <TranslatedText en="Using While Loop" bn="While লুপ ব্যবহার করে" />

<TranslatedText
  en={<p>The while loop is the most straightforward alternative to a for loop. It provides the same functionality with a slightly different syntax where you manually manage the counter variable.</p>}
  bn={<p>while লুপ for লুপের সবচেয়ে সহজ বিকল্প। এটি একটু ভিন্ন সিনট্যাক্সের সাথে একই কার্যকারিতা প্রদান করে যেখানে আপনি ম্যানুয়ালি কাউন্টার ভেরিয়েবল পরিচালনা করেন।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 0;
    while (i < 5) {
        cout << i << endl;
        i++;
    }
    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
i = 0
while i < 5:
    print(i)
    i += 1
```
  </Tab>
  <Tab value="JavaScript">
```javascript
let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}
```
  </Tab>
</Tabs>

## <TranslatedText en="Using Recursion" bn="রিকার্শন ব্যবহার করে" />

<TranslatedText
  en={<p>Recursion is a powerful technique where a function calls itself. It's particularly useful for tree traversal and divide-and-conquer algorithms, though it can replace simple loops as well.</p>}
  bn={<p>রিকার্শন একটি শক্তিশালী কৌশল যেখানে একটি ফাংশন নিজেকে কল করে। এটি বিশেষভাবে ট্রি ট্রাভার্সাল এবং ডিভাইড-এন্ড-কনকার অ্যালগরিদমের জন্য দরকারী, যদিও এটি সাধারণ লুপও প্রতিস্থাপন করতে পারে।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

void loop(int i, int max) {
    if (i >= max) return;
    cout << i << endl;
    loop(i + 1, max);
}

int main() {
    loop(0, 5);
    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
def loop(i, max_val):
    if i >= max_val:
        return
    print(i)
    loop(i + 1, max_val)

loop(0, 5)
```
  </Tab>
  <Tab value="JavaScript">
```javascript
function loop(i, max) {
  if (i >= max) return;
  console.log(i);
  loop(i + 1, max);
}

loop(0, 5);
```
  </Tab>
</Tabs>

## <TranslatedText en="Using Do-While Loop" bn="Do-While লুপ ব্যবহার করে" />

<TranslatedText
  en={<p>The do-while loop executes the code block at least once before checking the condition. This is useful when you need to ensure the loop body runs at least one time.</p>}
  bn={<p>do-while লুপ শর্ত পরীক্ষা করার আগে কোড ব্লকটি অন্তত একবার কার্যকর করে। এটি দরকারী যখন আপনাকে নিশ্চিত করতে হবে যে লুপ বডি অন্তত একবার চলে।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 0;
    do {
        cout << i << endl;
        i++;
    } while (i < 5);
    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
# Python doesn't have do-while, but we can simulate it
i = 0
while True:
    print(i)
    i += 1
    if i >= 5:
        break
```
  </Tab>
  <Tab value="JavaScript">
```javascript
let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);
```
  </Tab>
</Tabs>

## <TranslatedText en="Using Array Methods" bn="অ্যারে মেথড ব্যবহার করে" />

<TranslatedText
  en={<p>Modern programming languages provide functional programming methods that can iterate over collections without explicit loop syntax. These methods are more declarative and often more readable.</p>}
  bn={<p>আধুনিক প্রোগ্রামিং ভাষাগুলি ফাংশনাল প্রোগ্রামিং মেথড প্রদান করে যা স্পষ্ট লুপ সিনট্যাক্স ছাড়াই সংগ্রহের উপর পুনরাবৃত্তি করতে পারে। এই মেথডগুলি আরও ঘোষণামূলক এবং প্রায়শই আরও পাঠযোগ্য।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
#include <vector>
#include <algorithm>
using namespace std;

int main() {
    vector<int> numbers = {0, 1, 2, 3, 4};

    // Using for_each
    for_each(numbers.begin(), numbers.end(), [](int n) {
        cout << n << endl;
    });

    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
numbers = [0, 1, 2, 3, 4]

# Using map
list(map(print, numbers))

# Or using list comprehension
[print(n) for n in numbers]
```
  </Tab>
  <Tab value="JavaScript">
```javascript
const numbers = [0, 1, 2, 3, 4];

// Using forEach
numbers.forEach(n => console.log(n));

// Using map
numbers.map(n => console.log(n));
```
  </Tab>
</Tabs>

## <TranslatedText en="Using Goto Statement (Not Recommended)" bn="Goto স্টেটমেন্ট ব্যবহার করে (প্রস্তাবিত নয়)" />

<TranslatedText
  en={<p>While goto statements can technically implement loops, they are generally considered bad practice in modern programming due to reduced code readability and maintainability. This is shown here only for educational purposes.</p>}
  bn={<p>যদিও goto স্টেটমেন্ট প্রযুক্তিগতভাবে লুপ বাস্তবায়ন করতে পারে, কোড পাঠযোগ্যতা এবং রক্ষণাবেক্ষণযোগ্যতা হ্রাসের কারণে আধুনিক প্রোগ্রামিংয়ে এগুলি সাধারণত খারাপ অনুশীলন হিসাবে বিবেচিত হয়। এটি এখানে শুধুমাত্র শিক্ষামূলক উদ্দেশ্যে দেখানো হয়েছে।</p>}
/>

<Tabs items={['C++', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

int main() {
    int i = 0;

    loop_start:
    if (i < 5) {
        cout << i << endl;
        i++;
        goto loop_start;
    }

    return 0;
}
```
  </Tab>
  <Tab value="JavaScript">
```javascript
// JavaScript doesn't support goto
// This pattern is NOT recommended
let i = 0;
function loopWithLabel() {
  loopStart: while (true) {
    if (i >= 5) break loopStart;
    console.log(i);
    i++;
  }
}
loopWithLabel();
```
  </Tab>
</Tabs>

## <TranslatedText en="Using Tail Recursion (Optimized)" bn="টেইল রিকার্শন ব্যবহার করে (অপ্টিমাইজড)" />

<TranslatedText
  en={<p>Tail recursion is a special form of recursion where the recursive call is the last operation in the function. Some compilers can optimize tail recursion to avoid stack overflow issues.</p>}
  bn={<p>টেইল রিকার্শন হল রিকার্শনের একটি বিশেষ রূপ যেখানে রিকার্সিভ কল ফাংশনের শেষ অপারেশন। কিছু কম্পাইলার স্ট্যাক ওভারফ্লো সমস্যা এড়াতে টেইল রিকার্শন অপ্টিমাইজ করতে পারে।</p>}
/>

<Tabs items={['C++', 'Python', 'JavaScript']} groupId="language">
  <Tab value="C++">
```cpp
#include <iostream>
using namespace std;

void tailLoop(int i, int max) {
    if (i >= max) return;
    cout << i << endl;
    tailLoop(i + 1, max);  // Tail call
}

int main() {
    tailLoop(0, 5);
    return 0;
}
```
  </Tab>
  <Tab value="Python">
```python
def tail_loop(i, max_val):
    if i >= max_val:
        return
    print(i)
    return tail_loop(i + 1, max_val)  # Tail call

tail_loop(0, 5)
```
  </Tab>
  <Tab value="JavaScript">
```javascript
function tailLoop(i, max) {
  if (i >= max) return;
  console.log(i);
  return tailLoop(i + 1, max);  // Tail call
}

tailLoop(0, 5);
```
  </Tab>
</Tabs>
